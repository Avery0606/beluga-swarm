## 📦 安装

```bash
git clone https://github.com/Avery0606/beluga-swarm.git
cd beluga-swarm
npm run bootstrap
npm run setup-skills
```

添加到 `opencode.json`：
```json
{
  "plugin": ["file:///path/to/beluga-swarm/opencode-plugin/index.ts"]
}
```
